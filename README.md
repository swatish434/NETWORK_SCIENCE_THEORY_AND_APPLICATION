# NETWORK_SCIENCE_THEORY_AND_APPLICATION
Indian Stock Analysis on the NSE 500 data using the concepts covered in the Network Science Course 
The report contains an analysis of the Indian Stock market using NSE 500 data.

Objectives

Create stock correlation networks at multiple thresholds (0.6, 0.4, 0.2).

Construct a Minimum Spanning Tree (MST) to study the backbone structure.

Analyze the network using:

Degree and degree distribution

Centrality metrics (degree, betweenness, closeness, eigenvector)

Clustering coefficients

Assortative mixing

Similarity scores

Interpret results in the context of financial connectivity, sectoral relationships, and market hierarchy.
